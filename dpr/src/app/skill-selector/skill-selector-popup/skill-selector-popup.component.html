Timeline Builder Tips

Skill Quick Select

  • Your most recently placed skills appear on this list towards the left.
  • Use the keys 0-9 to quickly select skills.
  • Hold the Shift key while clicking a skill to lock it to the start of the list, or unlock it.

Timeline

  • Hold Control while placing skills to keep the current skill selected.
  • Hold Shift or drag from an empty slot to select multiple skills at once.
  • Press Shift+D to duplicate your currently selected skills.
